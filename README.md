## Commit Message Guidelines
| emoji              | type      | content                          |
| ------------------ | --------- | -------------------------------- |
| :page_facing_up:   | post:     | 新文章                           |
| :sparkles:         | feat:     | 新特性                           |
| :bug:              | fix:      | 修改问题                         |
| :recycle:          | refactor: | 代码重构                         |
| :pencil:           | docs:     | 文档修改                         |
| :lipstick:         | style:    | 代码格式修改, 注意不是 css 修改  |
| :white_check_mark: | test:     | 测试用例修改                     |
| :arrow_up:         | chore:    | 其他修改, 比如构建流程, 依赖管理 |
